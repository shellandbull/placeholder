# ActiveAgent Examples (Rails)

This is a Ruby on Rails application that hosts example integrations and demos for [ActiveAgent](https://www.activeagents.ai/).

## Getting Started

Nothing to see here

### Requirements
- Ruby 3.2+
- Rails 7+
- PostgreSQL (or your preferred database)

### Setup
```bash
git clone <repo-url>
cd <repo>
bundle install
bin/rails db:setup
bin/rails server
